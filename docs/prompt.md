Replay 제어용 ROS2 노드를 작성해 줘.
주요 기능은 다음과 같아.

* python 기반
* GUI 기반
* 주요 GUI 컴포넌트는 다음과 같아
  * 데이터베이스에 등록된 에피소드 목록을 보이는 영역
    * 에피소드 목록에서 한 에피소드를 지정하면 해당 에피소드를 replay 함
    * replay될 에피소드가 출력할 대상 토픽들을 선택할 수 있는 윈도우
  * /camera/image_raw 토픽에서 출력되는 이미지를 화면에 보이는 영역
  * "초기화" 버튼이 있어서 로봇을 특정 위치로 이동시키는 기능
    * move_group_client에서 제공하는 move_to_named_target를 사용함.
